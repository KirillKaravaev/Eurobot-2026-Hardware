*****************************************************MicroRos********************************************************************

Каталог содержит в себе 2 подкаталога - Hardware и Software, а также файлы MicroRos.cpp и MicroRos.h, содержащие саму реализацию 
microros. 

Hardware включает в себя классы, реализующие физическое взаимодействие программного обеспечения с периферией микроконтроллера, и
могут различаться для различных микроконтроллеров

Software представляет собой программное обеспечение, осуществляющее математическую обработку сигналов и реализующее различные алгоритмы,
эти классы универсальные и не зависят от выбранного микроконтроллера

История версий

V0.0.0 (23.04.2025)  - Копирован и видоизменен старый проект. (eurobot_2024_embedded). Содержит код для двигателей
                       JGB37-3650 (блок "kinematics" для преобразования оборотов в минуту в скорости вращения и др.,
                       блок "motors" для аппаратного управления моторами(выдача шим-импульсов для контроля скорости,

                       блок "imu" для вычисления скоростей и угла поворота, подключенный через i2c.

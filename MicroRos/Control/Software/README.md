Каталог содержит 2 подкаталога - Src и Inc. Представляет собой программное обеспечение, осуществляющее математическую обработку 
сигналов и реализующее различные алгоритмы, эти классы универсальные и не зависят от выбранного микроконтроллера

Src включает в себя исходный код .cpp для классов, реализующих математическую обработку сигналов или алгоритмы управления

Inc включает в себя хэдеры .h для исходного кода каталога Src

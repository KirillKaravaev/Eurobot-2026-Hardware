*****************************************************servo_ST3215******************************************************************

Библиотека для управления серводвигателями ST3215 и аналогами по полудуплексному UART.

Основана на библиотеке SCS. Сами сервоприводы управляются по протоколу dinamixel (похоже)
https://emanual.robotis.com/docs/en/dxl/protocol1/

Каталог содержит 2 подкаталога - Src и Inc. Включает в себя классы, реализующие физическое взаимодействие программного обеспечения 
с периферией микроконтроллера, и могут различаться для различных микроконтроллеров

Src включает в себя исходный код .cpp для классов, реализующих физическое взаимодействие с периферией микроконтроллера

Inc включает в себя хэдеры .h для исходного кода каталога Src

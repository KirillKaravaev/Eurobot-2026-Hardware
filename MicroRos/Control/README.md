Library for STM32 with mathematical function (Software) and functions for some peripherial control (Hardware)

Каталог содержит в себе 2 подкаталога - Hardware и Software

Hardware включает в себя классы, реализующие физическое взаимодействие программного обеспечения с периферией микроконтроллера, и
могут различаться для различных микроконтроллеров

Software представляет собой программное обеспечение, осуществляющее математическую обработку сигналов и реализующее различные алгоритмы,
эти классы универсальные и не зависят от выбранного микроконтроллера
